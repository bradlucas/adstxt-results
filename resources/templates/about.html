About

This project was developed by Brad Lucas. It is part of a larger effort to build a system to manage the results from crawling domains for Ads.txt files. This site in particular was built to host a small system which gathers the output of a Google Ads.txt crawler which returns domains which have Ads.txt files. The results are stored in a branch in a GitHub repository in files named by date. Retrieving these files programmatically was the initial reason in building the underlying library of this site.

The source code for this project are available in GitHub at the listed link below.
